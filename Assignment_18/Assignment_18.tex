\section{Problem}
Consider the vector space V of real polynomials of degree less than or equal to n. Fix distinct real numbers $a_0, a_1, \cdots, a_k$. For $p \in V$
\begin{align*}
    max\cbrak{\abs{p(a_j)}: 0\leq j \leq k}
\end{align*}
defines a norm on V when

\section{Solution}
A norm on vector space has the property that
\begin{align}
    &\norm{x} \ge 0 \\
    &\norm{x} = 0 \iff x = 0 \label{p2}
\end{align}
For $max\cbrak{\abs{p(a_j)}: 0\leq j \leq k}$ to be a norm on V it should satisfy \eqref{p2} that is, 
\begin{align}
    max\cbrak{\abs{p(a_j)}} = 0 \iff p(x) = 0 \label{norm}
\end{align}
i.e., $p(x)$ should be identically zero.
Also, 
\begin{align}
    \implies p(a_0) = 0\\
    p(a_j) = 0 \\
    \vdots \\
    p(a_k) = 0
\end{align}
Now, if $k<n$ then $p(x)$ can be given as,
\begin{align}
    p(x) = (x-a_0)(x-a_1)\cdots(x-a_k)q(x) \label{poly}
\end{align}
where $p(x)$ is of order m($\leq n$) and $q(x)$ is of the order (m-k-1).

Thus from \eqref{poly}, $p(x)$ is not identically zero. 

When $k \ge n$, then from \eqref{poly}, it can be seen that the order of $p(x)$ becomes $>n$ which is not possible as V is vector space of real polynomials with degree less than or equal to n.

Thus, for $k\ge n$, $p(x) = 0$ i.e., it is identically zero. Hence, $max\cbrak{\abs{p(a_j)}: 0\leq j \leq k}$ defines a norm on V only if $k \ge n$.
